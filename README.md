# 📊 Online Learning Analytics Dashboard (Power BI)

## 📌 Project Overview

This project presents an interactive **Power BI dashboard** analyzing an online learning platform.
It provides insights into **user engagement, course popularity, pricing, and revenue trends**.

The dashboard is designed to help stakeholders understand:

* Course performance
* User behavior
* Revenue generation
* Content strategy optimization



## 🎯 Objectives

* Analyze course distribution and growth over time
* Understand user engagement and subscriber trends
* Identify top-performing subjects and courses
* Evaluate pricing strategies and revenue patterns



## 🛠️ Tools & Technologies

* Power BI
* Data Cleaning & Transformation
* Data Visualization
* DAX (Data Analysis Expressions)



## 📂 Dataset

* Source: Online Learning Platform Dataset
* Features include:

  * Course Title
  * Subject
  * Price
  * Subscribers
  * Reviews
  * Published Year
  * Is Paid

## 📈 Key Insights

* Web Development courses have the highest subscriber base
* Paid courses generate significantly more revenue
* Engagement varies across subjects, with some niche categories underperforming
* Pricing has a direct impact on revenue but not always on subscriber count
